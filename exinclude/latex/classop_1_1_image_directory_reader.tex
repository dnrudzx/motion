\doxysubsection{상세한 설명}
\mbox{\hyperlink{classop_1_1_image_directory_reader}{Image\+Directory\+Reader}} is an abstract class to extract frames from a image directory. Its interface imitates the cv\+::\+Video\+Capture class, so it can be used quite similarly to the cv\+::\+Video\+Capture class. Thus, it is quite similar to \mbox{\hyperlink{classop_1_1_video_reader}{Video\+Reader}} and \mbox{\hyperlink{classop_1_1_webcam_reader}{Webcam\+Reader}}. 

image\+Directory\+Reader.\+hpp 파일의 14 번째 라인에서 정의되었습니다.
