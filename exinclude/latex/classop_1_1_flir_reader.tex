\doxysubsection{상세한 설명}
\mbox{\hyperlink{classop_1_1_flir_reader}{Flir\+Reader}} is an abstract class to extract frames from a F\+L\+IR stereo-\/camera system. Its interface imitates the cv\+::\+Video\+Capture class, so it can be used quite similarly to the cv\+::\+Video\+Capture class. Thus, it is quite similar to \mbox{\hyperlink{classop_1_1_video_reader}{Video\+Reader}} and \mbox{\hyperlink{classop_1_1_webcam_reader}{Webcam\+Reader}}.
