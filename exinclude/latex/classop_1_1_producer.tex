\doxysubsection{상세한 설명}
\mbox{\hyperlink{classop_1_1_producer}{Producer}} is an abstract class to extract frames from a source (image directory, video file, webcam stream, etc.). It has the basic and common functions (e.\+g., get\+Frame, release \& is\+Opened).
